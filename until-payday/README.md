# Until Payday
A script that will tell you how many days you have to wait until you get paid.

Create a `.paydayrc` in your home folder that contains the date you get paid:

```
PAYDAY_DATE=26
```

The run the script using Yarn or npm:

```
yarn start
```

# Build
```
npm run build
```

# Build + Watch

```
npm run watch
```
